# FITNESS-TRAINER-CRM

Простая CRM система для организации тренировок, записи клиентов к тренерам и управления оплатами в фитнес-сфере.

## Возможности

**Для клиентов:**
*   Регистрация и авторизация в системе
*   Просмотр доступных тренеров и их расписания
*   Запись на индивидуальные или групповые тренировки
*   Онлайн оплата тренировок
*   Просмотр истории своих тренировок и платежей

**Для тренеров:**
*   Управление своим расписанием тренировок
*   Подтверждение/отклонение записей клиентов
*   Просмотр статистики по тренировкам
*   Отслеживание заработка (еженедельно/ежемесячно)
*   Настройка доступных услуг и цен

**Для администратора:**
*   Полный контроль над пользователями и тренерами
*   Управление расписанием и бронированиями
*   Финансовая статистика и отчетность
*   Настройка тарифов и комиссий
*   Мониторинг всех транзакций

**Технические особенности:**
*   Простой и интуитивно понятный интерфейс
*   Безопасная система платежей
*   Напоминания о тренировках (email/telegram)
*   Базовая аналитика и отчеты
*   Мультиплатформенность (веб + мобильная адаптация)

## Установка

### Требования
*   Python 3.8 или выше
*   PostgreSQL 12+ или SQLite для разработки
*   Redis (для кэширования и очередей задач)

### Инструкция

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/yourusername/fitness-trainer-crm.git
    cd fitness-trainer-crm
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # Windows: venv\Scripts\activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте конфигурацию в `.env`:**
    ```env
    SECRET_KEY='ваш-секретный-ключ'
    DEBUG=True
    DATABASE_URL='postgresql://user:password@localhost/fitness_crm'
    REDIS_URL='redis://localhost:6379/0'
    EMAIL_HOST='smtp.gmail.com'
    EMAIL_PORT=587
    EMAIL_USER='ваш-email@gmail.com'
    EMAIL_PASSWORD='пароль-приложения'
    ```

5.  **Примените миграции базы данных:**
    ```bash
    python manage.py migrate
    ```

6.  **Создайте суперпользователя:**
    ```bash
    python manage.py createsuperuser
    ```

7.  **Запустите сервер разработки:**
    ```bash
    python manage.py runserver
    ```

8.  **Запустите Redis (для фоновых задач):**
    ```bash
    redis-server
    ```

## Основные функции

### Для клиентов:
*   **Регистрация/Вход** — Создание аккаунта или вход в систему
*   **Поиск тренеров** — Просмотр доступных тренеров с фильтрацией
*   **Запись на тренировку** — Выбор времени, тренера и типа тренировки
*   **Оплата онлайн** — Безопасная оплата через интегрированную систему
*   **История тренировок** — Архив всех пройденных тренировок
*   **Личный кабинет** — Управление профилем и настройками

### Для тренеров:
*   **Календарь тренировок** — Управление своим расписанием
*   **Подтверждение записей** — Принятие/отклонение запросов клиентов
*   **Мои клиенты** — Список текущих и постоянных клиентов
*   **Финансовый отчет** — Просмотр заработка за период
*   **Настройки услуг** — Управление типами тренировок и ценами

### Для администратора:
*   **Панель управления** — Обзор всей системы
*   **Управление пользователями** — Редактирование клиентов и тренеров
*   **Финансовый мониторинг** — Просмотр всех транзакций и комиссий
*   **Статистика** — Аналитика по посещаемости и доходам
*   **Настройки системы** — Конфигурация тарифов, комиссий, расписания

## Команды управления

```bash
# Создание миграций
python manage.py makemigrations

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск тестов
python manage.py test

# Сбор статических файлов
python manage.py collectstatic

# Запуск фоновых задач (Celery)
celery -A core worker -l info

# Запуск планировщика задач (Celery Beat)
celery -A core beat -l info
